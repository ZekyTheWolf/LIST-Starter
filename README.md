# Latest version
#### This will automaticly install all php dependencies & create necessary things for laravel
```
composer create-project zekythewolf/list-starter
```

## Install npm dependencies
```
npm install
```

## Running aplication
```
php artisan serve
npm run dev
```

### And you should be ready to go.
